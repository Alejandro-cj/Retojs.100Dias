Día 4: ¿Es par o impar?
Descripción:
Hoy vas a practicar cómo usar estructuras condicionales en JavaScript. Tu tarea es escribir un programa que determine si un número es par o impar.

Objetivo:
Utilizar condicionales (if y else).
Trabajar con el operador módulo (%).
Mostrar resultados en la consola.
Instrucciones:
Crea una función llamada esParOImpar que reciba un número como parámetro.
Usa una estructura condicional para determinar si el número es par o impar.
Imprime en la consola un mensaje indicando el resultado.

---------------------
Explicación:
Operador módulo %: Este operador devuelve el resto de dividir un número por otro. Si el resto de dividir un número entre 2 es 0, entonces el número es par.
Condicional: Usamos un if para comprobar si numero % 2 === 0. Si es verdadero, el número es par; de lo contrario, es impar.
Imprimir resultado: Usamos console.log() para mostrar el resultado de la evaluación.
Tarea adicional:
Prueba números negativos y verifica que el programa funcione correctamente.
Agrega validación para comprobar si el valor ingresado es realmente un número (puedes usar typeof).
Extiende la función para manejar casos especiales como el número 0 (que es par).
